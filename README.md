
# DOWN

```sh
$ npm i -g down
```

Don't hesistate to fork/issues/bugreport/pull request.

# USAGE

```sh
$ down gh
   UP [github] Everything operating normally.
   UP [github] Everything operating normally.
   ...
```

or

```sh
$ down hv
   UP [harvest] last response time: 609
   UP [harvest] last response time: 609
   ...
```
